# CAAI Next.js Frontend (TypeScript)

This is a Next.js dashboard to control and interact with your FastAPI backend agents.

## Configure

Create a `.env.local` in this folder:

```
NEXT_PUBLIC_BACKEND_URL=http://localhost:8000
```

## Run

```
npm install
npm run dev
```

Then open http://localhost:3000

## Build

```
npm run build
npm start
```
